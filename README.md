#### 🕵️ [Vestige](https://github.com/th4r4s/vestige)

> <b>Traces people leave behind, across the live web and its archive.</b>

An OSINT identifier finder that mines <b>emails, phones, social profiles, crypto wallets and map locations</b> from a domain's current site and its Wayback Machine history. The live site gets cleaned up; the archive still remembers.

✅ &nbsp; Checksum validated crypto (BTC · ETH · TRON · XRP · TON), pure Python<br/>
📞 &nbsp; Region aware phones with carrier and line type enrichment<br/>
🧩 &nbsp; Reads JS rendered (SPA) content without a browser<br/>
🌐 &nbsp; Trilingual UI (EN · RU · TR) and a tamper evident audit log
